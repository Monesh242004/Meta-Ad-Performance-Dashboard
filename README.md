Facebook & Instagram Advertising Analysis Dashboard

📊 Facebook & Instagram Advertising Analysis

A Power BI dashboard project that analyzes advertising campaigns across Facebook and Instagram platforms. The project focuses on campaign performance, audience targeting, demographics, budget allocation, and advertisement strategies using interactive visualizations.

📌 Project Overview

This project provides insights into digital advertising campaigns by combining user demographics, campaign information, and advertisement targeting data. The dashboard helps marketers understand audience behavior, optimize campaigns, and improve advertising performance.

🎯 Objectives
Analyze Facebook and Instagram advertising campaigns.
Understand user demographics and interests.
Evaluate campaign budgets and durations.
Examine advertisement targeting strategies.
Compare platform-wise advertising distribution.
Generate actionable insights for marketing decisions.
📂 Dataset Information

The project uses three datasets:

1. Users Dataset (users.csv)

Contains user demographic information:

User ID
Gender
Age
Age Group
Country
Location
Interests
2. Campaign Dataset (campaigns.csv)

Contains campaign-related information:

Campaign ID
Campaign Name
Start Date
End Date
Duration (Days)
Total Budget
3. Ads Dataset (ads.csv)

Contains advertisement targeting information:

Ad ID
Campaign ID
Ad Platform (Facebook/Instagram)
Ad Type
Target Gender
Target Age Group
Target Interests
