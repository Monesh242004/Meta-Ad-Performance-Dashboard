# 📊 Facebook & Instagram Advertising Analysis Dashboard

## 📖 Project Description

This Power BI project analyzes advertising campaigns conducted on Facebook and Instagram. The dashboard provides insights into user demographics, campaign budgets, advertisement targeting, and platform performance. It helps marketers understand audience behavior and optimize advertising strategies through interactive visualizations.

---

## 🎯 Project Objectives

- Analyze advertising campaigns across Facebook and Instagram.
- Understand audience demographics and interests.
- Compare advertisement distribution across platforms.
- Evaluate campaign budgets and durations.
- Identify target audience patterns.
- Generate data-driven marketing insights.

---

## 🛠️ Tools & Technologies Used

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- CSV Datasets

---

## 📂 Dataset Information

### Users Dataset (`users.csv`)
Contains demographic information about users.

**Columns:**
- User ID
- Gender
- Age
- Age Group
- Country
- Location
- Interests

### Campaigns Dataset (`campaigns.csv`)
Contains information about marketing campaigns.

**Columns:**
- Campaign ID
- Campaign Name
- Start Date
- End Date
- Duration
- Total Budget

### Ads Dataset (`ads.csv`)
Contains advertisement details and targeting information.

**Columns:**
- Ad ID
- Campaign ID
- Platform
- Ad Type
- Target Gender
- Target Age Group
- Target Interests

## 🔄 Data Preparation

### Data Cleaning
- Checked for missing values.

Relationships were created between:
